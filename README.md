
# Quadrotor Trajectory Simulation

**Author:** Emmanuel Agboro  
**Context:** Build project (class project)  

This project implements a **quadrotor simulation** with trajectory tracking using a **PID controller**. The simulation demonstrates quadrotor dynamics, controller design, trajectory generation, and visualization in Python.
